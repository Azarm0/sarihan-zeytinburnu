# Sarıhan İşkembe Zeytinburnu — Website Mockup

## Quick test

1. Unzip this folder.
2. Double-click `index.html` to open it in a modern browser.

For the most reliable result, run a tiny local web server in this folder:

### Python

```bash
python3 -m http.server 8080
```

Then visit: http://localhost:8080

### Node.js

```bash
npx serve .
```

## Package notes

- Restaurant and menu photographs are bundled under `assets/menu/` or embedded directly in `index.html`.
- Google Maps and Google Fonts require an internet connection.
- The menu is interactive: category tabs and search run entirely in the browser.
- This is a design mockup, not the restaurant's official website.
- Menu prices were sourced from Sarıhan's public online menu; confirm current Zeytinburnu prices with the restaurant.
